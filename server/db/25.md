# 在 postgres 中，查询时如何对 jsonb 数据格式化



::: tip Issue 
 欢迎在 Issue 中讨论: [Issue 25](https://github.com/shfshanyue/Daily-Question/issues/25) 
:::

使用 `jsonb_pretty` 函数，示例如下

``` sql
> select jsonb_pretty('{"a": {"b": 4}}'::jsonb)
+----------------+
| jsonb_pretty   |
|----------------|
| {              |
|     "a": {     |
|         "b": 4 |
|     }          |
| }              |
+----------------+
SELECT 1
Time: 0.018s
```
