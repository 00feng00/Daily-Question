# 什么是隔离级，都有哪些隔离级



::: tip Issue

:::

::: tip Author 
回答者: [shfshanyue](https://github.com/shfshanyue) 
:::

隔离级事关并发事务的隔离机制，ANSI SQL 中定义了四种隔离级，分别是

+ Read Uncommited
+ Read Committed
+ Repetable Read
+ Serializable
