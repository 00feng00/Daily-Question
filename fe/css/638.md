:::

::: tip Author

+ `:hover`
+ `:active`
+ `:first-child`
+ `:last-child`
+ `:nth-child`
+ `:first-of-type`
+ `:last-of-type`
+ `:nth-of-type`
+ `:empty`
+ `:check`
+ `:disable`
