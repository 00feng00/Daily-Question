# HTML 标签有哪些行内元素

常见的标签有以下几种，可参考 [inline element](https://developer.mozilla.org/en-US/docs/Web/HTML/Inline_elements)

+ a
+ img
+ picture
+ span
+ input
+ textarea
+ select
+ label
