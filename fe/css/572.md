## 使用 Flex

注意使用 `calc(100% / 3)` 设置 `flex-basis`，同时它的缺陷是**不能够很好的为子元素设置左右间隙**

``` scss
.flex-container {
  display: flex;
  flex-wrap: wrap;
  // gap: 1rem;
  
  
  .item {
   flex: 0 0 calc(100% / 3);
  }
}
```

## 使用 Grid

使用 Grid 直接操作容器即可，对于子元素之间的间隙也可以很好地控制，可看出 Grid 更为简单

``` scss
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 1rem;
}
```
