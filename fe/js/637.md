# 使用 JS 如何生成一个随机字符串



::: tip Issue 
 欢迎在 Issue 中交流与讨论: [Issue 637](https://github.com/shfshanyue/Daily-Question/issues/637) 
:::

::: tip Author

``` js
const random = () => Math.random().toString(36).slice(2)

random()
// => "c1gdm25ezch"
random()
// => "oir5ppqmar"
```
