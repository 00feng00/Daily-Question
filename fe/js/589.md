# 关于 this，判断以下代码输出

::: tip 更多描述 
 ``` js
function foo() {
  console.log( this.a );
}

var a = 2;

(function(){
  "use strict";

  foo();
})();
``` 
:::

输出: 2

**只有在存在 this 的函数中设置严格模式**，this 为 undefined。因此此时会正常输出。
