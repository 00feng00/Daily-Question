# React 中，cloneElement 与 createElement 各是什么，有什么区别



::: tip Issue 
 欢迎在 Issue 中讨论: [Issue 22](https://github.com/shfshanyue/Daily-Question/issues/22) 
:::

首参不一样，直接上 API

``` js
React.cloneElement(
  element,
  [props],
  [...children]
)

React.createElement(
  type,
  [props],
  [...children]
)
```
