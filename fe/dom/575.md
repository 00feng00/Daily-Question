# Data URL 的应用场景及如何生成

Data URLs 由四个部分组成：

1. 前缀(data:)
2. 指示数据类型的MIME类型
3. 如果二进制数据则为可选的base64标记，比如图片
4. 数据


```
data:[<mediatype>][;base64],<data>
```
